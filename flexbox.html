1.Describe the main differences between the CSS Flexbox layout model and the CSS Grid layout model. When would you choose to use one over the other?

Ans .Flexbox is a one-dimensional layout model ideal for arranging items in a row or column, perfect for simple, linear designs like navbars. Grid is two-dimensional, allowing control over both rows and columns, making it better for complex layouts like dashboards or full pages. Use Flexbox for flexibility in smaller components, and Grid for structured, larger-scale layouts.

2.Explain the role of the following key properties in the Flexbox layout model
  1. justify content
  2. align-items
  3. gap
  4. flex-direction
  5. flex-wrap

Ans.

  1. justify content- align items along the main axis
  2. align-items- align items along the cross axis
  3. gap- gives space between flex-items
  4. flex-direction- sets the direction of the main axis
  5. flex-wrap-allows items to wrap to the next line if needed
